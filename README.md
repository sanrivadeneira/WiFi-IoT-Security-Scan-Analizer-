# WiFi-IoT-Security-Scan-Analizer-
 📡 RICKRSA IoT Auditor - ESP32-S3 (Heltec V3)  Este proyecto es una herramienta de auditoría de redes inalámbricas diseñada específicamente para la placa *Heltec WiFi Kit 32 V3 (ESP32-S3)*. Permite el monitoreo en tiempo real de redes WiFi y dispositivos Bluetooth Low Energy (BLE) a través de un Dashboard Web dinámico.
